# Buss的个人博客

在线体验：https://www.tutupersonalblog.top

#### 介绍
VUE 3轻量级个人博客，作者：图图大王

#### 技术栈
- 前端：Vue 3 + vite + pinia，组件库使用 Element Plus
- 部署：前端静态化后托管至 Gitee Pages

#### 项目亮点
 - 完整的用户认证（登录/注册/退出 + Token + 路由守卫）
 - 文章管理（Markdown编辑器 + 增删改查 + 分页）
 - 个人中心（头像上传、昵称修改、密码修改含原密码验证）
 - 用户体验细节（密码强度、二次确认、加载状态）

#### 安装教程

1. 克隆仓库
   git clone https://gitee.com/图图大王/Buss的个人博客.git
2. 安装依赖
   cd vue3-personalfourn-admin
   pnpm install
3. 配置环境变量
4. pnpm dev
5. 访问 http://localhost:3000

#### 使用说明

1.  需要登录进入，默认用户【id：图图；密码：123456】。或者自行注册后自行跳转
2.  可阅读、编辑文章
3.  可修改个人信息

#### 声明

1.  没有正经的后端接口，使用的模拟数据库和token
